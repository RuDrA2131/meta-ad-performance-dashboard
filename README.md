# 📱 Meta Ad Performance Dashboard Analysis

### Power BI Marketing Analytics Project

**Power BI | DAX | Data Modeling | Data Visualization**

---

## 📌 Project Overview

Digital advertising generates massive amounts of campaign data across multiple platforms. This Power BI project analyzes Meta (Facebook & Instagram) advertising performance to help marketers monitor campaign effectiveness, understand audience behavior, optimize budgets, and improve conversion outcomes through data-driven decision-making.

---

## ❗ Problem Statement

Marketing teams often face challenges in measuring campaign success across multiple ad formats and audience segments.

| Challenge               | Description                                                               |
| ----------------------- | ------------------------------------------------------------------------- |
| 📊 Campaign Monitoring  | Tracking key advertising metrics across multiple campaigns and platforms  |
| 🎯 Audience Insights    | Understanding demographic segments driving the highest conversions        |
| 💰 Budget Optimization  | Identifying high-performing campaigns for efficient budget allocation     |
| 📈 Performance Analysis | Measuring CTR, Engagement Rate, Conversion Rate, and Purchase Performance |

---

## 🎯 Project Objectives

| #  | Objective                     | Description                                                                  |
| -- | ----------------------------- | ---------------------------------------------------------------------------- |
| 01 | Campaign Performance Analysis | Analyze impressions, clicks, engagements, and purchases across campaigns     |
| 02 | Audience Segmentation         | Evaluate purchasing behavior by age, gender, and location                    |
| 03 | Conversion Optimization       | Track CTR, Conversion Rate, and Purchase Rate to improve campaign efficiency |

## 📊 Dashboard Features

### 1️⃣ KPI Performance Overview

A comprehensive overview of Meta advertising performance metrics, enabling marketers to monitor campaign success and business outcomes in real time.

* Tracks Impressions, Clicks, Shares, Comments, Purchases, and Engagements
* Measures CTR, Engagement Rate, Conversion Rate, and Purchase Rate
* Provides instant visibility into overall campaign performance
* Supports data-driven marketing decisions

### 2️⃣ Audience Analysis

Detailed analysis of customer demographics to understand purchasing behavior and engagement patterns.

* Purchases by Gender
* Purchases by Age Group
* Audience Segmentation Insights
* Identification of high-converting customer segments

### 3️⃣ Geographic Performance Analysis

Interactive map visualization showing purchase activity across different countries and regions.

* Country-wise purchase distribution
* Geographic performance comparison
* Regional customer engagement analysis

### 4️⃣ Trend Analysis Dashboard

Monitor campaign performance trends over time.

* Weekly Purchase Trends
* Hourly Purchase Trends
* Monthly Performance Analysis
* Seasonal and behavioral pattern identification

### 5️⃣ Ad Type Performance Analysis

Compare the effectiveness of different Meta ad formats.

* Video Ads Performance
* Stories Ads Performance
* Image Ads Performance
* Carousel Ads Performance
* CTR, Engagement Rate, and Conversion Comparison

## 🔑 Key Insights

### 📈 Campaign Performance

| Metric            | Value  |
| ----------------- | ------ |
| Total Impressions | 216K   |
| Total Clicks      | 25.4K  |
| Total Engagements | 29.3K  |
| Total Purchases   | 1.3K   |
| CTR               | 11.76% |
| Engagement Rate   | 13.56% |
| Conversion Rate   | 5.21%  |
| Total Budget      | $2.5M  |

### 🎯 Audience Insights

* Female users generated the highest number of purchases.
* Users aged 24–32 showed the strongest purchasing activity.
* Multiple geographic regions contributed significantly to campaign performance.
* Engagement and conversion rates varied across ad formats.

### 📊 Ad Performance Insights

* Stories Ads generated the highest impressions.
* Video Ads delivered strong engagement rates.
* Carousel Ads maintained competitive conversion performance.
* Ad format selection significantly impacted campaign outcomes.
